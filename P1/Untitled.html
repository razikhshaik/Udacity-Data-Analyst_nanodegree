(Validate the presence of stroop effect)

Question 1: What is our independent variable? What is our dependent variable?

Answer: Independent variable : Condition - Congruent/Incongruent (Word and color combination) Dependent Varible: time taken to interpret ink colors

Question 2: What is an appropriate set of hypotheses for this task? What kind of statistical test do you expect to perform? Justify your choices.

Answer: Null - Hypothesis : mean time taken for congruent task is equal to the mean time taken for incongruent task μcongruent = μincongruent Alternative - Hypothesis : mean time taken for congruent task is less than the mean time taken by incongruent task μcongruent < μincongruent μcongruent represents the population mean response time for congruent task μincongruent represents the population mean response time for congruent task Test to perform: one-tailed Dependent T-Test for Paired Samples - 1) I would go with t-test because we are given only the sample statistics with a sample size less than 30 and no estimates of the population parameters. Because of this the sampling distribution of the sample means have a t distribution (thicker tails). 2) To explain the presence of stroop effect we have to compare the means of congruent task and incongruent task sample which are related together as both are measure of reaction time. 3) Stroop effect says that it takes more time to interpret a incongruent task and is prone to errors than a congruent task, so one-tailed test should be enough to tell if stroop effect exists or not. Reason for one tailed test: Stroop effect(defn from wikipedia) It is clearly mentioned that naming incongruent task takes longer time and is more prone to error.

x̄congruent = 14.05 x̄incongruent = 22.016 n = sample size = 24 point estimates are dependent upon the sample estimates (considering the sample estimates to be good representation of population parameters) point estimate (mD) = x̄congruent - x̄incongruent = 14.05 - 22.015 = -7.96 t-statistic = (x̄congruent - x̄incongruent)/(s/√n) degree of freedom = n-1 = 23 s = standard deviation of sample differences = 4.864 t-statistic = -7.96/(4.864/√24) = -8.0207 for confidence 95%, α = 0.05 one tailed test, t-critical value = -1.714 p value = 0.00000002 Confidence Interval for 95% (-∞, -7.96 + (1.714*(4.864/√24)),∞) = (-∞, -6.25) This interval doesn't contain 0 which means we are 95% confident that time taken to solve incongruent task for population will be greater than or equal to (time taken by incongruent + 6.25 secs) Applying the sample statistics to population parameters (μcongruent - μincongruent) <= -6.25 μincongruent >= μcongruent + 6.25 t-statistic << t-critical p value << α (0.05) It is highly unlikely to have selected a sample with p value = 0.00000002 just by chance, this shows the effect as descrubed as stroop effect. Therefore, we can reject Null Hypothesis. Time taken for congruent task is less compared to congruent task. Yes results match my expectation.

When there is a mismatch it takes time to read the word and also interpret the color of the word (which is not the same as the word). Brain is doing just doing 1 task when there is no mismatch, as the associations for color to word are already made in brain before. Brain has to do 2 parallel tasks when there is a mismatch, the associations that it previously has result is conflict because of the mismatch, Brain takes extra time to resolve this conflict to confidently process the final answer.
